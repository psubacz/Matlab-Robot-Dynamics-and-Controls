Sensor Ex Client

Matlab client for Sensor Ex application.

Description

The 'Sensor Ex Client' block fetches data from a connected mobile device.

To use this block, copy it anywhere in your model select Device Number and specify the appropriate Image Size vector.

Some points to keep in mind when using the block:

  • The block operates correctly in real time simulation. Try using Real-Time Pacer block.

The Sensors block works on all supported Simulink platforms.

Parameters

Device Number

Device number of the mobile device.

Image Size

1-D array image size of the form [Height Width].

Inputs and Outputs

Inputs

None.

Outputs

  1. IP Local address. (1-D array)
  2. IP Device address. (1-D array)
  3. Sensors values. (1-D array)
  4. R red channel of the image. (1-D array)
  5. G green channel of the image. (1-D array)
  6. B blue channel of the image. (1-D array)
