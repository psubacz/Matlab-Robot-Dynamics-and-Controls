Sensors Values Manipulator

Manipulate sensors values fed by a Sensor Ex Client block.

Description

The 'Sensors Values Manipulator' block manipulates sensors values fed by a Sensor Ex Client block. The block's output ports can be changed via the Output Type and Output Time Stamp parameters.

To use this block, copy it anywhere in your model and connect the Sensor Ex Client sensors output to its input.

Some points to keep in mind when using the block:

  • The block requires a Sensor Ex Client in the simulink model.

The Sensors block works on all supported Simulink platforms.

Parameters

Output Type

Defines block output ports.

  • Standard: adds 2 output ports Acceleration, Orientation.

    1. Acceleration 1-D array with acceleration values around X, Y and Z respectively.
    2. Orientation 1-D array with orientation values around X, Y and Z respectively.


  • Divided Arrays: adds 6 output ports Acceleration around X, Y and Z, Orientation around X, Y and Z.

    1. AccX acceleration value around X.
    2. AccY acceleration value around Y.
    3. AccZ acceleration value around Z.

    4. OriX orientation value around X.
    5. OriY orientation value around Y.
    6. OriZ orientation value around Z.

Output Time Stamp

Time needed to fetch sensors values in seconds.
On
Adds TimeStamp output port.
Off
Removes TimeStamp output port.

TimeStamp may vary depending on the device sample rate and processing speed.

Inputs and Outputs

Inputs

None.

Outputs

  1. TotalTime sum of device's timestamps in seconds. (Double)
More outputs depending on Output Type and Output Time Stamp parameters check above.
